# A simple chess game made from scratch in python

### Why I did this
I wanted to get better at Python. I have already made a TicTacToe game with Tkinter but I wanted to do a more challenging game.


To play, just run the .py file, or the .exe if you are on windows...

If you are on pyroid (python interpreter on android), change "self.button_size = 60" to "self.button_size = 130" (around line 60)
